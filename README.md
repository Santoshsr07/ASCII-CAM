# 📸 ASCII Cam — HD Engine

A high-performance **real-time ASCII webcam renderer** built using pure HTML, CSS, and JavaScript.

Convert your live camera feed into stunning ASCII art with advanced controls, filters, and export features — all running directly in the browser.

## 🚀 Features

### 🎥 Real-Time ASCII Rendering
- Live webcam feed converted into ASCII characters
- Smooth rendering with FPS control

### ⚡ Performance Presets
- ⚡ Performance (fastest)
- ⚖ Balanced (default)
- 🔍 HD (high quality)
- 💎 Ultra (maximum detail)

### 🎛️ Advanced Controls
- Brightness, contrast, gamma, threshold
- Edge detection & sharpening
- ASCII density and character set customization

### 🎨 Color Modes
- Grayscale
- Full color
- Matrix (green)
- Amber
- Cyan
- Red heat
- Custom color picker

### 🪞 Effects & Tools
- Mirror view
- Invert colors
- Bold ASCII rendering
- Flip camera (front/back)
- Zoom and font size adjustment

### 📸 Capture & Export
- Save output as PNG image
- Export ASCII as `.txt`
- Record ASCII frames

### 📱 Responsive UI
- Clean modern interface
- Works on desktop and mobile

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Glassmorphism + modern UI)
- Vanilla JavaScript
- Canvas API
- WebRTC (`getUserMedia`)
